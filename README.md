# 雙語書房 · jp-biz

中文 ⇄ 日本語的單頁翻譯工具，提供：

- 中文草稿轉成可直接整理寄出的日文商業郵件
- 中文→日文郵件完成後可自動商務潤稿（預設開啟，可在設定關閉）
- 日文來信翻成繁體中文
- 日文來信可一鍵「日文潤飾」成自然的商務日文（不翻譯、保留原意）
- 面對面中日口譯與譯文朗讀
- 反譯、商務潤稿、字彙表與選字問 AI
- 預設使用 GPT-5.6 Terra，並可在設定切換 GPT-5.6 Luna

本專案為純前端靜態頁面。OpenAI API Key 與選用的 Google Cloud TTS Key 由使用者在瀏覽器本機設定。
